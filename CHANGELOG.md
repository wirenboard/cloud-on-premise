# Changelog

Все значимые изменения в проекте фиксируются в этом файле.

## [1.2.0] - 2026-05-29

### Добавлено

- Возможность переопределить тему и тело письма-приглашения в организацию через переменные окружения `INVITE_EMAIL_SUBJECT` и `INVITE_EMAIL_BODY`. Если переменные не заданы (или пусты), используется встроенная локализация по языку приглашающего. В теле письма поддерживаются переводы строки через `\n`; ссылка-приглашение всегда добавляется в конец тела.

### Изменено

- Автоматизирован релизный pipeline: workflow «Start Release» одной кнопкой создаёт ветки `onprem-release/X.Y.Z` сразу в пяти репозиториях стека, бампает `VERSION` и добавляет запись в `CHANGELOG`.
- Добавлен workflow «Deploy to test server» для развёртывания тестовых образов на dev-сервере по SSH одной кнопкой из GitHub Actions.
- Релизный workflow `make_release.yml` переведён на встроенный `GITHUB_TOKEN` с явными `packages: write` правами вместо ручного PAT, устранены ошибки авторизации в GHCR. Из набора файлов GitHub Release исключены пустые конфиги `tls/.gitkeep` и `jwt/`, ломавшие публикацию.

### Исправлено

- Фронтенд: корректное определение локали браузера (для нестандартных значений выбирается ближайшая поддерживаемая) и стабильная отправка заголовка `Accept-Language` в запросах к бэкенду — устраняет ситуации, когда письма приходили на неожиданном языке.

## [1.1.1] - 2026-05-27

### Добавлено

- Локализация писем-приглашений в организацию: для англоязычных получателей письмо приходит на английском (определяется по заголовку `Accept-Language`).

## [1.1.0] - 2026-04-17

### Добавлено

- Добавлена возможность подменять фронтенд-ассеты: логотип на странице, favicon и иконки, используемые во вкладках браузера.

## [1.0.0] - 2026-01-17
